# Hotel Booking System
A simple object oriented Java project that simulates a hotel booking system.

# Features
- Room types (Single, Double, Suite)
- Book a room
- Cancel booking
- View all rooms
- View available rooms
- View all bookings
- Price calculation using polymorphism

# Concepts Used
- Inheritance
- Polymorphism
- Encapsulation
- ArrayList
- Object-oriented design

# How it works
The system manages rooms and bookings. Each room type has its own pricing logic. Guests can book and cancel rooms through a console menu.

# Technologies
- Java
- OOP
